# Premier League Fixtures Dashboard

A simple and interactive web application built with **React**, **Tailwind CSS**, and **shadcn/ui**, which fetches and displays **Premier League fixtures** along with detailed **event timelines** including goals, cards, penalties, and more.

### Using NPM

```bash

git clone https://github.com/AnaniyaBelew/SportsHubTestProject.git

# Navigate into the project folder
cd SportsHubTestProject

# Install dependencies
npm install

# Start the development server
npm run dev
```
### Using Docker
```bash

docker build -t sports-hub .

# Run the Docker container
```
